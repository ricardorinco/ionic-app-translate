# App Translate

This is the repository of the App Translate (Ionic/Angular 5) application

## Installation

Install the dependencies and start the application.

### angular and ionic

``` sh
    npm install -g @angular/cli
    npm install -g ionic cordova
```

### root

```sh
    cd viacep
```

### dependencies

```sh
    npm install
```

### run

```sh
    ionic serve
```

### browser

```sh
http://localhost:8100
```
